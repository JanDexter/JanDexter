## About

I am a Cloud and Software Engineer, Computer Science student, and digital artist. I specialize in designing cloud systems from first principles, building full-stack web applications, and creating intuitive user experiences. My engineering philosophy is rooted in pragmatic problem solving. I am tool-agnostic and do not adhere to fixed tech stacks; I evaluate and select the right technology for each problem and adapt quickly to new environments and languages. My system architecture choices are driven by project needs rather than defaulting to the stack I'm most familiar with. Beyond writing code, I explore 3D modeling, pixel art, and webtoons, which allows me to bridge the gap between robust backend architecture and thoughtful frontend design. I also actively lead student tech communities and organize hackathons to empower the next generation of developers.

- 🎓 BS Computer Science @ **Ateneo de Davao University**
- ☁️ Cloud Club Lead (AWS) · EVP @ **ACCESS**
- 🛠️ DevOps Engineer @ **SYSDEV** · Freelance since 2020
- 🗣️ Speaker — IT Week 2026, PWA Pilipinas, Build Forward 2025
